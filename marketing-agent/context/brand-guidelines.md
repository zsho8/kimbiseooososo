- 카라칼(동물)에서 영감 — 빠르고 민첩한 움직임, 예민한 성격 → 스포츠 부상에 빠르게 반응하고 날카롭게 피드백하는 브랜드

### 브랜드 BI 의미
- **눈(정사각+곡선)**: 카라칼의 눈으로 스포츠 문제를 예리하게 바라봄

## 언어·카피 스타일

### 문체 및 톤
- 기본 문체: ~합니다 / ~입니다 (공식), ~요 (SNS 캐주얼)
- 타깃 호칭: 선수님 / 플레이어 / 여러분
- 페르소나: **전문적이지만 운동을 진심으로 사랑하는 동료 운동인**

### 핵심 메시지 프레임
- "테이프 없이" → "착용만으로" → "관절을 보호"
- "물리치료사가 설계한" → 신뢰·전문성 강조
- 수치 강조 (항상 구체적으로):
  - 연간 비용 85% 절감
  - 100회 세탁 후 92% 이상 유지
  - 와디즈 1,055% 달성
  - 네이버 4.9점 (238건)
  - 피부 자극도 0%

### 권장 표현
- 부상 예방 / 퍼포먼스 향상 / 해부학 기반 / 메디컬 등급
- WE SUPPORT ALL THE PLAYERS
- 모든 플레이어를 위한 / 내 몸을 아는 양말
- 10초 착용 / 셀프 테이핑

### 금지 표현
- 근거 없는 "최고·1등·완치" 표현
- 의학적 치료 효과 직접 주장 ("치료합니다", "통증이 사라집니다")
- 경쟁사 직접 언급·비방
