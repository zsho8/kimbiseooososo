# 카라칼(CARACAL) 디자인 스타일 가이드

## 전체 디자인 철학

카라칼은 **메디컬 신뢰성 + 스포츠 역동성**을 동시에 표현해야 한다.
경쟁사 분석(CEP·Bauerfeind·2XU·Compressport) 기준 스포츠 메디컬 브랜드의 황금 법칙:

| 디자인 원칙 | 적용 방식 |
|------------|---------|
| 다크 배경 = 프리미엄·메디컬 신뢰 | 기술 설명, 제품 단독 샷, IR 자료 |
| 라이트 배경 = 라이프스타일·일상 | 커뮤니티 콘텐츠, SNS 피드 |
| 수치는 가장 크게 | "92%", "1,055%", "4.9점" — 텍스트보다 3~4배 크게 |
| 여백은 넉넉하게 | 슬라이드 전체 여백 최소 60px |
| 슬라이드당 메시지 1개 | 텍스트 5줄 이하 엄수 |

## 브랜드 컬러 시스템

### 확인된 제품 컬러 (실물 기준)
- 제품 라인: **PRO BLACK WHITE TAPING** / **PRO WHITE BLACK TAPING**
- 결론: 블랙 ↔ 화이트 고대비 구조가 브랜드 DNA

### 컬러 팔레트

| 역할 | 컬러 | 코드 | 사용처 |
|------|------|------|--------|
| Primary Dark | 블랙 | `#1A1A1A` | 다크 배경, 메인 텍스트 |
| Primary Light | 화이트 | `#FFFFFF` | 라이트 배경, 반전 텍스트 |
| Neutral | 미드그레이 | `#6B6B6B` | 부제목, 캡션 |
| Light BG | 라이트그레이 | `#F5F5F5` | 뉴스레터, 정보 카드 배경 |
| Brand Accent | **[포인트 컬러 — 미확인]** | `#??????` | CTA, 수치, 뱃지 강조 |

> 포인트 컬러는 브랜드 BI 파일 확인 후 입력. 미입력 시 임시 대안:
> - 에너지/퍼포먼스 느낌: `#E8500A` (딥 오렌지)
> - 야생/역동 느낌: `#C8962A` (앰버 골드)
> - 메디컬/신뢰 느낌: `#1B4F8A` (딥 네이비)

### 경쟁사 컬러 벤치마크 (참고용)
| 브랜드 | 메인 컬러 | 악센트 | 배경 전략 |
|--------|---------|--------|---------|
| CEP | 레드 `#E13E3E` + 네이비 `#275CAB` | 라임그린 `#A6CC39` | 화이트 배경 선호 |
| Bauerfeind | 다크 앤스러사이트 + 네이비 | 토스카나 레드 | **다크 배경 — 프리미엄 느낌** |
| 2XU | 블랙 + 네이비 | 형광핑크, 라임, 오렌지 | 혼합 (스포츠 장면 따라) |
| Compressport | 다이나믹 고명도 | 밝은 대비색 | 액션 사진 배경 |

## 카드뉴스 (인스타그램/쓰레드)

### 사이즈
- 정방형: **1080 × 1080 px** (기본)
- 세로형: **1080 × 1350 px** (저장율 높음, 권장)

### 슬라이드 구성 — 카라칼 표준 (7장)

| # | 역할 | 핵심 요소 | 카라칼 적용 예시 |
|---|------|----------|----------------|
| 1 | 커버 | 후킹 제목 + CARACAL 로고 | "발목 부상 없이 마라톤 완주하는 법" |
| 2 | 문제 공감 | 페인포인트 (비용·불편·부작용) | "테이프 비용만 연 50만원? 붙이기도 어렵고" |
| 3 | 해결책 제시 | CARACAL 포지셔닝 | "착용만 해도 테이핑 효과 — 물리치료사가 설계" |
| 4 | 기술 근거 | 수치 강조 (크게) | **"100회 세탁 후 92% 유지"** |
| 5 | 사회적 증거 | 리뷰·성과 데이터 | **"네이버 4.9점 · 와디즈 1,055%"** |
| 6 | 가격 비교 | 연간 비용 절감 | **"연 50만원 → 6만원, 85% 절감"** |
| 7 | CTA | 구매 유도 + 브랜드 태그 | "지금 스마트스토어에서 → 링크 바이오" |

### 레이아웃 규칙
- 헤드라인: 60~80px, ExtraBold
- 수치 강조: 120~160px, 화면 중앙 배치
- 본문: 30~36px, Regular
- 마진: 60px (상하좌우 균일)
- 최대 텍스트: 슬라이드당 5줄 이하 엄수

### 배경 선택 기준
- 기술 설명 슬라이드 → **다크 배경 (#1A1A1A)** + 화이트 텍스트
- 리뷰/커뮤니티 슬라이드 → **화이트 배경** + 다크 텍스트
- CTA 슬라이드 → 브랜드 포인트 컬러 배경
