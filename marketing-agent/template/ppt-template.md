# PPT 템플릿 마크다운

> 이 파일은 PPT를 매번 이미지로 참조하는 대신 MD로 저장해 토큰을 절약하기 위한 템플릿입니다.
> 디자인 에이전트는 이 MD를 참조해 `anthropic-skills:pptx` 스킬로 PPTX를 생성합니다.

---

## PPT 기본 규격
- 슬라이드 크기: 33.87 × 19.05 cm (와이드 16:9)
- 폰트: [제목용 폰트] / [본문용 폰트]
- 배경 기본색: [컬러 코드]

---

## 마스터 레이아웃

### Layout 1 — 커버 슬라이드
```
배경: 풀 블리드 이미지 또는 브랜드 그라데이션
┌─────────────────────────────────────┐
│                                     │
│   [메인 제목]                       │
│   폰트: [폰트명] 44pt Bold           │
│   컬러: 흰색                        │
│                                     │
│   [서브타이틀]                      │
│   폰트: [폰트명] 24pt Regular        │
│                                     │
│   [발표자] | [날짜]                 │
│   [브랜드 로고 — 우하단]            │
└─────────────────────────────────────┘
```

### Layout 2 — 섹션 구분
```
배경: 브랜드 메인 컬러 단색
┌─────────────────────────────────────┐
│   [섹션 번호]  01 / 02 / 03         │
│                                     │
│   [섹션 제목]                       │
│   폰트: [폰트명] 36pt Bold 흰색      │
└─────────────────────────────────────┘
```

### Layout 3 — 본문 (기본)
```
┌─────────────────────────────────────┐
│ [슬라이드 제목] — 28pt Bold         │
├─────────────────────────────────────┤
│                                     │
│  본문 내용                          │
│  폰트: [폰트명] 18pt Regular         │
│  최대 5줄 / 줄당 40자               │
│                                     │
│  ● 포인트 1                         │
│  ● 포인트 2                         │
│                                     │
└─────────────────────────────────────┘
```

### Layout 4 — 2단 레이아웃
```
┌──────────────────────────────────┐
│ [슬라이드 제목]                  │
├─────────────────┬────────────────┤
│   [좌측 내용]   │  [우측 내용]   │
│                 │                │
└─────────────────┴────────────────┘
```

### Layout 5 — 마무리 슬라이드
```
배경: 브랜드 컬러
┌─────────────────────────────────────┐
│                                     │
│   [마무리 제목 또는 감사 인사]      │
│                                     │
│   [CTA 또는 연락처]                │
│   [웹사이트] | [이메일]            │
│                                     │
│   [브랜드 로고 — 중앙]             │
└─────────────────────────────────────┘
```

## 시각 요소 가이드

### 아이콘
- 스타일: [라인형 / 필드형 / 컬러형]
- 크기: [N]pt 기준
- 컬러: 브랜드 컬러 계열

### 차트/그래프
- 기본 컬러: 브랜드 팔레트 사용
- 폰트: [폰트명] [크기]pt
- 범례 위치: 우측 또는 하단

### 이미지
- 비율: 16:9 또는 4:3
- 스타일: [예: 실사 / 일러스트 / 플랫 디자인]
- 처리 방식: [예: 오버레이 30% 적용]
