# 유튜브 썸네일 템플릿

## 사이즈 규격
- 표준: 1280 × 720 px (16:9)
- 저장 형식: JPG 또는 PNG
- 최대 파일 크기: 2MB

---

## 썸네일 유형별 템플릿

### 유형 A — 인물 + 텍스트형
```
[레이아웃]
┌─────────────────────────────────┐
│  [제목 텍스트]     [인물 사진]  │
│  최대 25자                      │
│                                 │
│  [서브 텍스트]                  │
└─────────────────────────────────┘
```
- 배경: [단색 / 그라데이션]
- 인물: 오른쪽 배치, 시선은 텍스트 방향
- 텍스트: 굵은 폰트, 고대비 컬러

---

### 유형 B — 텍스트 중심형
```
[레이아웃]
┌─────────────────────────────────┐
│                                 │
│     [메인 제목]                 │
│     [부제목]                    │
│                                 │
│  [브랜드 로고]                  │
└─────────────────────────────────┘
```
- 배경: 브랜드 컬러 또는 관련 이미지
- 텍스트 크기: 제목 80-100pt, 부제목 40-50pt

---

### 유형 C — 비포/애프터형
```
[레이아웃]
┌────────────┬────────────────────┐
│  BEFORE    │    AFTER           │
│  [이미지]  │    [이미지]        │
└────────────┴────────────────────┘
```

---

## 썸네일 텍스트 작성 규칙

### 제목 (메인)
- 최대 글자 수: 15-20자
- 스타일: [예: "N가지 방법" / "절대 하지 마세요" / "몰랐던 사실"]
- 피해야 할 표현: 클릭베이트성 과장

### 서브 텍스트
- 최대 글자 수: 20-25자
- 역할: 제목 보완 또는 구체화

---

## 컬러 & 폰트

### 컬러 조합
| 배경 | 제목 텍스트 | 서브 텍스트 |
|------|------------|------------|
| #[코드] | #[코드] | #[코드] |
| 흰색 | 브랜드 메인 | 다크 그레이 |
| 브랜드 메인 | 흰색 | 밝은 계열 |

### 권장 폰트
- 한국어 제목: [폰트명] ExtraBold
- 영문 제목: [폰트명] Black
