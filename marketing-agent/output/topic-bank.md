# 카라칼 인스타 카드뉴스 주제 뱅크 (자동 발행 순환 큐)

> 자동 스케줄(월·목)이 이 파일의 맨 위 `대기` 주제를 하나 꺼내 카드뉴스를 만든 뒤,
> 상태를 `발행대기(검수)`로 바꾸고 캡션·파일 경로를 기록한다.
> 주제가 3개 미만으로 떨어지면 자동 실행이 웹 리서치로 새 주제 5개를 보충한다.
> 모든 주제는 "정보 먼저 → 카라칼은 보조" 원칙. 의학적 단정 금지.

## 상태 범례
`대기` → `발행대기(검수)` → `승인/게시완료` (또는 `보류`)

---

> 주제 선정 원칙: **인스타그램 전반의 최근 트렌드(계정 무관)를 후킹으로 얹고, 본문은 카라칼이 잘하는 부상예방·관절보호 정보 → 테이핑 양말 보조 연결.**
> 트렌드 근거: `instagram-trend-report.md` 참조. 상위 트렌드 = Hyrox / Zone2 저강도 러닝 / 러닝크루·인증샷 / 오래하는 운동(롱제비티) / 부상예방 튜토리얼.
