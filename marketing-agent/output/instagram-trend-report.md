## 3. 카라칼 적용 결론 (콘텐츠 제작 룰)
- 카드뉴스(캐러셀)는 **저장·공유 트렌드에 최적** → 현재 전략 유지·강화.
- **8~12장**으로 정보 밀도 있게(너무 간결 X). 각 장 = 헤드라인 + 실질 정보 1덩이(수치·체크리스트).
- **마지막 장에 '저장+공유' 명시 유도**(예: "저장하고 러닝메이트에게 공유해요").
- **신뢰도 = 공유력**: 과장 대신 솔직한 단서(예: "주 6h 미만이면 고강도도 효율적")를 넣으면 전문성·공유 모두↑.
- 사진은 **사진(위)+텍스트 패널(아래) 분리형**으로 가독성 확보(글씨가 사진과 겹치지 않게).
- 주제는 **트렌드 키워드(Hyrox·Zone2·러닝크루·롱제비티)를 후킹으로 얹고**, 본문은 카라칼이 잘하는 **부상예방·관절보호 정보 → 테이핑 양말 보조 연결**.

## 4. 브랜드 빌딩 미션 (장기)
카드뉴스는 단발 게시물이 아니라 **카라칼 브랜드를 멋지게 키워가는 핵심 자산**이다.
매 발행마다: ①전문성(근거·수치) ②브랜드 일관성(공식 주황 #E44E12·로고·톤) ③저장·공유 설계를
지켜 누적 신뢰와 팬덤을 쌓는다. "WE SUPPORT ALL THE PLAYERS"를 콘텐츠로 증명한다.
- 톤은 과한 광고 대신 정보 우선·'저장각'.
